Exercício Final: Projeto Site de Clínica — Guia Completo

Este documento descreve detalhadamente o exercício solicitado (criar um site "quase" completo de uma clínica médica), explica o que deve ser feito, por que cada etapa importa e traz exemplos práticos e modelos para você copiar e adaptar.

Visão geral do desafio

Objetivo: criar um site institucional simples de uma clínica (você escolhe a especialidade) aplicando os conceitos do módulo:

  • Estruturação e formatação de texto
  • Mídias (imagens, iframe)
  • Formulários
  • Tabelas (thead, tbody, tfoot, th, td)
  • Navegação entre páginas (links internos)

Resultado esperado

Um mini-site com quatro páginas principais, todas seguindo o mesmo template visual (menu, header, content, footer):

  • index.html — Página Principal
  • sobre.html — Sobre a clínica
  • servicos.html — Horários e serviços (com tabela)
  • contato.html — Formulário, telefones e mapa (iframe)

Template base (modelo) — como organizar cada página

Criar um arquivo template.html só para referência ajuda a manter padrão. Exemplo mínimo de estrutura que você deve replicar em cada página:

<!DOCTYPE html>
<html lang="pt-br">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Nome da Página - Clínica X</title>
