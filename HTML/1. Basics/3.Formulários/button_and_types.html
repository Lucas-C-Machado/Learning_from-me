Botões e seus Tipos

Neste arquivo você vai entender como funcionam os botões dentro de um formulário HTML. Todos os exemplos estão explicados passo a passo e com comentários no próprio código.

Campos do Formulário







Botão tipo "button"

Este tipo de botão NÃO envia o formulário. Ele serve apenas para executar ações através de JavaScript. Os alertas que você verá só funcionam porque usamos JavaScript nos eventos.

• onclick — dispara ao clicar



• onmouseover — dispara ao passar o mouse



• ondblclick — dispara ao clicar duas vezes



• onmouseout — dispara ao sair com o mouse



• onfocus — dispara ao focar no botão (corrigido)

Observação: se o evento mostrar um alert() **sem remover o handler**, o fechamento do alerta pode devolver o foco ao botão e disparar o evento novamente. Para evitar isso removemos o atributo onfocus antes de chamar o alert().



Botão tipo "reset"

Este botão limpa todos os campos do formulário.



Botão tipo "submit"

Envia o formulário. Como estamos usando method="GET", os dados aparecem na URL após o envio.



Mais Tipos de Botões

• <input type="button"> — versão antiga, ainda usada



• <input type="submit"> — envia o formulário



• <input type="reset"> — limpa o formulário



• Botão desabilitado



• Botão que envia para outro lugar (formaction)



• Botão que muda o método (formmethod)
