Como este formulário funciona:

  • Checkbox permite escolher várias opções ao mesmo tempo.
  • O atributo name="opcional[]" envia todas as escolhas dentro de um array.
  • disabled bloqueia a interação (ótimo para itens indisponíveis).
  • Radio permite escolher apenas UMA opção dentro do mesmo grupo (mesmo name).
  • O botão "Submit" envia tudo via método GET, aparecendo na URL.

Casos de uso:

  • Checkbox → opcionais de pizza, preferências extras, múltiplas seleções.
  • Radio → escolha única como "Sim/Não" ou seleção de bebida.
