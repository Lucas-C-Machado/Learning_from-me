Menu Lateral em HTML

Neste arquivo você verá como criar um menu lateral simples utilizando apenas HTML e estilos aplicados diretamente em cada elemento (sem usar arquivo CSS externo). Tudo está comentado passo a passo para facilitar o entendimento.


Como este menu foi construído

1. Estrutura básica: criamos um elemento <div> chamado menu-lateral, que funciona como a “caixa” do menu.

2. Estilos aplicados diretamente: como solicitado, todo o CSS foi colocado no atributo style de cada elemento, sem usar o <head>.

3. Lista de navegação: utilizamos uma lista <ul> para representar os itens do menu, pois é a forma mais semântica e correta de criar menus.

4. Remoção dos marcadores: menus não usam os pontos da lista, então removemos com list-style: none;.

5. Estilização dos links: deixamos os links mais visíveis retirando o sublinhado e aplicando cor azul e negrito.

6. Comentários didáticos: todas as partes do código possuem explicações detalhadas para auxiliar no aprendizado dos alunos.
