Elemento <video>

A tag <video> é usada para exibir vídeos diretamente no HTML, sem necessidade de plugins externos. Possui atributos como:

  • controls: adiciona botões de play, pausa e volume;
  • autoplay: tenta iniciar automaticamente;
  • width/height: controla o tamanho do vídeo.


Exemplo da tag <span>

A tag <span> serve para destacar ou estilizar apenas um trecho do texto dentro de um parágrafo, sem quebrar a linha.

Porém, o <span> sozinho não altera nada visualmente. Isso acontece porque ele é apenas um elemento inline genérico, criado para marcar uma parte do texto. Para realmente destacar, você precisa aplicar CSS (cor, tamanho, peso da fonte etc.). Ou seja:

  • <span> → marca o trecho
  • CSS → deixa o trecho destacado (cor, fonte, estilo...)

Lorem ipsum

O que é?

Lorem ipsum é um texto fictício usado como preenchimento (placeholder) em layouts. Com ele, você pode testar, criar e visualizar sites, verificando design, espaçamento, tipografia e estrutura antes do conteúdo real estar pronto.

Como criar Lorem Ipsum na IDE (VS Code)

Ao usar o VS Code, você pode gerar texto Lorem Ipsum automaticamente usando Emmet. Basta digitar os comandos abaixo e pressionar TAB:

  • lorem → gera um texto padrão
  • lorem20 → gera um texto com 20 palavras
  • lorem*3 → gera 3 parágrafos de Lorem Ipsum
  • p*5>lorem15 → cria 5 parágrafos, cada um com 15 palavras

Esses comandos não aparecem no site final, eles servem apenas para facilitar a criação de conteúdo fictício durante o desenvolvimento.
