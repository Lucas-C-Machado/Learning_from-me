Elemento <font>

O que é?

A tag <font> era usada antigamente no HTML para alterar cor, fonte e tamanho do texto. Porém, essa tag foi oficialmente descontinuada e não deve ser mais utilizada.

Por que não usar <font>?

Existem três motivos principais:

  • Não é semântica, não adiciona significado ao conteúdo.
  • Quebra a separação entre estrutura e estilo, mistura HTML com aparência.
  • Foi substituída por CSS, que é muito mais flexível e moderno.

Exemplo de código usando <font> (NÃO recomendado)

Este é o código original, para fins didáticos: 

<font color="red" face="Trebuchet MS, Tahoma">
    Este é um trecho do meu arquivo font
</font>

Como fazer da forma correta (usando CSS)

Hoje, o recomendado é usar CSS, seja em classes, IDs ou estilos inline.

Como seria no VS Code?

Basta criar um arquivo HTML normal e substituir qualquer uso de <font> por classes CSS, assim: 

span.texto-formatado
TAB

Isso criará automaticamente um elemento com classe para você estilizar.
