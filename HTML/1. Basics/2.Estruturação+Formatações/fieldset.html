Fieldset e Legend

O que é o <fieldset>?

A tag <fieldset> é usada para agrupar campos de um formulário, criando uma separação lógica e visual entre seções. Isso deixa o formulário mais organizado e melhora a experiência do usuário.

Para que serve?

  • Organizar blocos de informações;
  • Apresentar seções distintas (Dados Pessoais, Endereço, etc.);
  • Ajudar na acessibilidade, pois leitores de tela entendem melhor o agrupamento;
  • Melhorar a aparência e a estrutura de um formulário.

O que é a tag <legend>?

A tag <legend> define um título para o grupo de campos criado pelo fieldset. Sem o legend, o agrupamento perde significado visual e semântico.

Atributos úteis do <fieldset>

  • disabled: desativa todos os campos dentro do fieldset;
  • form: associa o fieldset a um formulário específico;
  • name: identifica o grupo.
