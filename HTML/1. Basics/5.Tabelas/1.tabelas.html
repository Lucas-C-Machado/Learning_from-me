Resumo Fácil para o Aluno

A estrutura básica de uma tabela é:

  • <table> → cria a tabela
  • <tr> → cria uma linha
  • <td> → cria uma célula comum (dados)
  • <th> → cria um cabeçalho (título das colunas)

A tabela nada mais é do que uma sequência de linhas (<tr>) que contém células (<td> ou <th>). Isso torna os dados fáceis de visualizar e organizar.
