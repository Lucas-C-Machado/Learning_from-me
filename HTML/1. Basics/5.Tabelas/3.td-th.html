Tags td e th

As tags td e th são utilizadas para criar células dentro das linhas (tr) de uma tabela HTML.

Diferença entre td e th

  • td: representa uma célula comum, usada para dados.
  • th: representa uma célula de cabeçalho, geralmente exibida em negrito e centralizada.

Exemplo Prático

Nome Idade
Fernanda 23
Tatiana 42
Marcelo 44

Explicação visual do exemplo acima

  • Na primeira linha, foram utilizadas células th, pois representam títulos da tabela.
  • As linhas seguintes utilizam td para representar os valores reais (dados).
  • O CSS aplica:
    • Cor especial no cabeçalho (th).
    • Listras alternadas nas linhas, facilitando a leitura.
    • Efeito hover ao passar o mouse.

Boas Práticas

  • Sempre utilize th para cabeçalhos, não td.
  • Mantenha a tabela organizada com títulos claros.
  • Utilize summary para acessibilidade (leitores de tela).
  • Evite tabelas para layout; use apenas para dados tabulares.
