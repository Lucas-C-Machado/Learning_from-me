Estilização de Tabelas no HTML

Neste arquivo, vamos focar exclusivamente em como aplicar estilos CSS para deixar tabelas mais organizadas, legíveis e visualmente agradáveis. A estrutura básica da tabela já está pronta, e o objetivo principal é entender como cada seletor e propriedade influencia no resultado final.

Aqui você aprenderá a:

  • Aplicar bordas e sombras;
  • Estilizar cabeçalho, corpo e rodapé;
  • Criar efeito zebrado em linhas;
  • Adicionar destaque ao passar o mouse (hover);
  • Melhorar a leitura e aparência geral da tabela.

Estas tags não alteram o funcionamento da tabela por si só, mas ajudam na estruturação do conteúdo e facilitam a aplicação de estilos via CSS.

Explicação do Processo

Para criar este arquivo voltado à estilização, utilizamos apenas HTML e CSS puro, mantendo tudo simples e direto para que os alunos compreendam facilmente a funcionalidade das tags. A estilização foi aplicada diretamente no elemento <style> no cabeçalho, para facilitar a visualização e evitar arquivos externos.

O objetivo principal deste arquivo é ensinar a função de cada parte da tabela e mostrar como o CSS pode ser aplicado para melhorar a leitura e organização dos dados.
