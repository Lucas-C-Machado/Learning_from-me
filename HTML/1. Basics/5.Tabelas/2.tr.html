Tag tr

A tag tr é usada dentro de uma tabela para representar uma linha. Cada tr contém células (td ou th) que formam o conteúdo da tabela.

Como funciona

  • tr cria uma nova linha na tabela.
  • td representa células com dados.
  • th representa células de cabeçalho.
  • Você pode estilizar cada linha usando CSS (como hover, cores, etc.).

Exemplo Prático

N° Nome Completo Idade
1 Lucas 22
2 Fernanda 23
3 Tatiana 42
4 Marcelo 44

Explicação do CSS aplicado

  • nth-child(even): aplica cor de fundo azul em linhas pares, facilitando a leitura (efeito zebra).
  • tr:hover: ao passar o mouse sobre a linha, ela muda para verde.
  • border-collapse: collapse: junta as bordas para visual mais limpo.
