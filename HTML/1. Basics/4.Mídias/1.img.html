Exemplo de SVG no HTML



Como este documento funciona:

  • <img> é uma tag auto-fechante usada para exibir imagens.
  • alt descreve a imagem caso ela não carregue (essencial para acessibilidade).
  • title exibe uma dica ao passar o mouse.
  • width redimensiona mantendo a proporção.
  • SVG permite criar imagens vetoriais dentro do próprio HTML.
