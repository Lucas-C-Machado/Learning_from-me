Exemplo de Uso da Tag <iframe>

O que este arquivo demonstra?

Este documento ensina de forma prática como funciona a tag <iframe>, incluindo:

  • Como incorporar vídeos externos via iframe
  • Como incorporar mapas (Google Maps)
  • Atributos importantes de segurança
  • Como o iframe funciona e seus casos de uso
  • Estrutura organizada para aprendizado didático

O que é um <iframe>?

O elemento <iframe> permite carregar uma página HTML externa dentro da sua própria página. Em outras palavras, ele cria uma “janela” que exibe outro conteúdo.

Alguns usos comuns incluem:

  • Vídeos do YouTube
  • Mapas como Google Maps
  • Formulários do Google Forms
  • Widgets externos
  • Documentos incorporados de outras plataformas

Segurança:
O iframe possui atributos que ajudam a controlar permissões, privacidade e o que o conteúdo incorporado pode ou não fazer, como allow, referrerpolicy e loading.


Exemplo 1 — Incorporando um vídeo do YouTube

Aqui incluímos um vídeo externo usando a URL de embed fornecida pelo próprio YouTube. Os atributos mais importantes são:

  • width / height — define o tamanho do vídeo
  • src — URL do vídeo incorporado
  • frameborder — remove bordas
  • allow — libera permissões (autoplay, clipboard, etc.)
  • allowfullscreen — habilita modo tela cheia
  • referrerpolicy — controla informações enviadas ao site


Exemplo 2 — Incorporando um mapa do Google Maps

O Google Maps fornece um código pronto para incorporar mapas. Este iframe contém várias informações:

  • Coordenadas da cidade
  • Zoom e ajuste da câmera
  • loading="lazy" — carrega somente quando necessário
  • allowfullscreen — permite tela cheia
  • referrerpolicy — protege dados do usuário


Como funciona tecnicamente o <iframe>?

  • <iframe> carrega outra página dentro da atual.
  • src contém o link do conteúdo incorporado.
  • allow controla recursos liberados ao conteúdo.
  • loading="lazy" melhora o desempenho.
  • referrerpolicy controla os dados enviados.
  • allowfullscreen ativa o modo tela cheia.

Casos de uso reais:

  • Vídeos incorporados (YouTube, Vimeo)
  • Mapas interativos
  • Formulários externos
  • Dashboards e gráficos via ferramentas externas
  • PDFs e documentos
