Exemplo de Uso da Tag <video> com Legendas

Player de Vídeo com Legendas



Como funciona este vídeo:

  • controls: exibe os botões padrão do player.
  • source: permite fornecer o vídeo em vários formatos.
  • track: adiciona legendas acessíveis (.vtt).
  • O navegador escolhe automaticamente o formato compatível.
  • poster: carrega uma miniatura antes da reprodução.
  • Fallback: mensagem caso o navegador seja muito antigo.

Como eu gerei as legendas deste vídeo:

Para produzir o arquivo de legendas .vtt deste vídeo, segui exatamente os passos abaixo:

  1. Enviei o vídeo para um software de transcrição automática, que converteu o áudio completo para texto no formato .txt.
    Ferramenta usada: Cockatoo – Transcrição Automática
  2. Copiei o texto transcrito e trouxe para o ChatGPT, pedindo para:
    • converter o texto em um arquivo .vtt sincronizado;
    • formatar no padrão WebVTT oficial.
  3. Existem softwares pagos com legenda automática sincronizada perfeitamente (como Whisper Pro, ASC, Descript, Premiere Auto-Captions etc.), mas como o objetivo aqui é didático — aprender como funciona a estrutura e aplicar em HTML — não houve necessidade de nenhuma ferramenta paga.
  4. Após gerar o arquivo final, salvei como:
    legenda_video.vtt
  5. Por fim, adicionei o arquivo .vtt ao meu projeto e utilizei a tag <track> dentro do elemento <video> para ativar as legendas no navegador.

Casos de uso

  • Vídeos educacionais com acessibilidade.
  • Páginas de treinamento corporativo.
  • Vídeos institucionais.
  • Apresentações de sistemas web.
