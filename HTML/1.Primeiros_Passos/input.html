1. Seleção de Arquivo (um arquivo)

Permite que o usuário escolha um único arquivo do computador.

<input type="file">



2. Seleção de Vários Arquivos (multiple)

Permite selecionar vários arquivos ao mesmo tempo usando o atributo multiple.

O usuário pode selecionar vários arquivos usando CTRL ou SHIFT.

<input type="file" multiple>



3. Campo Invisível (hidden)

Envia informações ocultas no envio de dados: ID, tokens, configurações internas, etc.

Não aparece na página, mas o valor existe no código.

<input type="hidden" name="idUsuario" value="4521">

(O campo hidden não aparece visualmente.)


4. Botão Enviar (submit)

Cria um botão que envia os dados de um formulário.

<input type="submit" value="Enviar">
