1. Estrutura Básica do HTML

Voltar ao topo

Todo arquivo HTML começa com uma estrutura padrão, composta por:

  • <!DOCTYPE html> - informa que é um documento HTML5;
  • <html> - elemento raiz da página;
  • <head> - contém informações invisíveis, como título e metadados;
  • <body> - contém todo o conteúdo visível para o usuário.

3. Parágrafos e Formatação de Texto

Voltar ao topo

O elemento básico de texto é o <p>. Também conhecido como Parágrafo.

Negrito (<strong> ou <b>) destaca partes importantes.

Sublinhado (<u>) pode indicar termos relevantes.

Marcação (<mark>) ajuda a chamar atenção para trechos específicos.

Texto pequeno (<small>) pode ser usado para notas.

Exemplo de texto com subscrito (<sub>) e sobrescrito (<sup>).

"Este é um exemplo de <blockquote>, usado para citações maiores."

Texto riscado representa algo removido ou substituído (<del>).

6. CSS Aplicado Diretamente no HTML (Inline)

Voltar ao topo

Podemos aplicar estilos direto na tag usando o atributo style.

Exemplo de sintaxe:

<p style="color: red;">Texto</p>

Este texto está azul.

Este texto está vermelho.

Este texto está verde.
